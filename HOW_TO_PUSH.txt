┌───────────────────────────────────────────────────────────────────────────┐
│ 🎯 EASIEST METHOD - Use the Helper Script                                │
└───────────────────────────────────────────────────────────────────────────┘

1. Get a Personal Access Token:
   → Go to: https://github.com/settings/tokens
   → Click "Generate new token (classic)"
   → Name it: "BioCare Deploy"
   → Check: ✅ repo (all permissions)
   → Generate and COPY the token

2. Run the helper script:
   
   Windows (PowerShell):
   → Right-click push-to-github.ps1
   → Select "Run with PowerShell"
   → Paste your token when prompted
   
   OR
   
   Windows (Command Prompt):
   → Double-click push-to-github.bat
   → Paste your token when prompted

3. Done! ✅

┌───────────────────────────────────────────────────────────────────────────┐
│ 💻 MANUAL METHOD - Command Line                                          │
└───────────────────────────────────────────────────────────────────────────┘

1. Get your token (same as above)

2. Open terminal in the project folder

3. Run these commands (replace YOUR_TOKEN):

   git remote set-url origin https://YOUR_TOKEN@github.com/BiocareAnalysis/Biocare.git
   git push -u origin main

4. Done! ✅
